# Решение судоку с помощью backtracking

Этот проект содержит реализации алгоритма решения судоку на языках C++, Python и Java, используя метод **backtracking** (обратный откат).

## Описание задачи

Цель задачи — заполнить пустые клетки в сетке судоку (обычно 9x9, где пустые клетки обозначены 0 или '.') числами от 1 до 9 так, чтобы выполнялись следующие правила:

1.  Каждая строка должна содержать цифры от 1 до 9 без повторений.
2.  Каждый столбец должен содержать цифры от 1 до 9 без повторений.
3.  Каждый из 9 субблоков 3x3 должен содержать цифры от 1 до 9 без повторений.

## Алгоритм (Backtracking)

Решение основано на рекурсивном методе **backtracking**, как описано в разделе 5.1 методических рекомендаций. Основные шаги алгоритма:

1.  **Найти пустую клетку:** Ищется первая клетка, содержащая 0 (или другое обозначение пустой клетки).
2.  **Базовый случай:** Если пустых клеток больше нет, значит, судоку успешно решено. Возвращается `true`.
3.  **Перебор:** Для найденной пустой клетки перебираются числа от 1 до 9.
4.  **Проверка валидности (is_valid):** Для каждого числа проверяется, можно ли его разместить в текущей клетке, не нарушая правил судоку (проверка строки, столбца и 3x3 блока).
5.  **Сделать выбор (make_choice):** Если число валидно, оно помещается в клетку.
6.  **Рекурсивный вызов:** Вызывается функция решения для оставшейся части судоку.
7.  **Проверка результата рекурсии:**
    *   Если рекурсивный вызов возвращает `true`, это означает, что решение найдено. Функция также возвращает `true`.
    *   Если рекурсивный вызов возвращает `false`, текущий выбор числа не привел к решению.
8.  **Откат (undo_choice / backtrack):** Если текущий выбор не worked, клетка снова помечается как пустая, и цикл перебора пробует следующее число.
9.  **Возврат false:** Если ни одно из чисел от 1 до 9 не приводит к решению из текущего состояния, функция возвращает `false`, сигнализируя, что предыдущий выбор был неверным.

## Как использовать

Каждый файл содержит пример начального заполнения судоку. Запустив соответствующий файл на языке программирования. Программа выведет исходное судоку и, если решение существует, решенное судоку. В противном случае, будет выведено сообщение, что решение не найдено.

## Сложность

*   **Временная сложность:** В худшем случае может быть экспоненциальной, O(...), в зависимости от количества пустых клеток и структуры головоломки.
*   **Пространственная сложность:** O(1), если не учитывать стек рекурсии. Глубина рекурсии может достигать количества пустых клеток.
